 Student Management System with Analytics

 Overview
This project is a web-based Student Management System built using PHP and MySQL.  
It manages student records, subjects, grades, archives, and includes an analytics dashboard for decision-making.

---

 Features

Students Module
- Add, edit, view, and archive students

 Subjects Module
- Manage subjects
- Add and archive subjects

 Grades Module
- Assign grades to students
- Compute averages and remarks
- Link students and subjects

 Archive Module
- Stores deleted records
- Restore functionality for students, subjects, and grades

 Analytics Dashboard
- Students per course
- Students per year level
- Average grades per subject
- Performance ranking (top and at-risk students)

 Export System
- CSV export of student records and analytics

---

 Technologies Used
- PHP
- MySQL
- HTML / CSS
- Git & GitHub

---

 How to Run
1. Install XAMPP
2. Import `studentrecordsV1.sql` into MySQL
3. Place project in `htdocs`
4. Start Apache & MySQL
5. Open `localhost/student_system`

---

 Developer Notes
This system demonstrates both OLTP (transaction processing) and OLAP (data analytics) concepts for academic evaluation purposes.
